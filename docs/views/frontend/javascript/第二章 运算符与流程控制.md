##### 38.字符串的截取

```javascript
let name = 'Meng';
# 截取第几个
console.log(name.slice(1));
console.log(name.substring(1));
console.log(name.substr(1));

# 第二参数，前两个是第一参数->第二参数
console.log(name.slice(1,3));
console.log(name.substring(1,3));
# 截取从1开始n个
console.log(name.substr(1,3));

substring不能从-1等负数截取
```
